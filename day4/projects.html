BOL - AI Public Speaking Coach

problem:Many students fear public speaking. There are very few simple tools that give instant feedback after a speech. BOL solves this problem.


solution: Users record their speech. The application analyzes Confidence, Speech Length, Pauses, Delivery, Speaking Style and generates feedback.

features:

  1. speech recording
  2. confidence score
  3. performance dashboard
  4. history tracking
  5. Improvement Suggestions
  6. Daily Practice Tasks
  7. Responsive UI
  8. progress graph

tech stakes:

  • css
  • javascript
  • webaudio api
  • gemini flash 2.5 lite
